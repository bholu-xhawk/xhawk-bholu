# Next.js 14 + Tailwind Starter

A minimal Next.js 14 (App Router) TypeScript project with Tailwind CSS and a simple homepage composed of Hero, About Me, and Projects sections, plus a site-wide Footer.

## Quick Start

- Prerequisites: Node.js 18+ (20+ recommended)
- Install dependencies: `npm install`
- Run dev server: `npm run dev`
- Build: `npm run build`
- Start production server: `npm run start`

## Project Structure

- `app/` - App Router pages and layout
- `components/` - Reusable UI components (Hero, AboutMe, Projects, Footer)
- `data/` - Simple data source for Projects

## Notes

- Tailwind CSS is set up via `postcss.config.js` and `tailwind.config.ts`
- External images from `via.placeholder.com` are allowed in `next.config.ts`
- TypeScript configuration is in `tsconfig.json` with strict mode enabled
